¿Que lenguaje usamos?

Usamos estos lenguajes para optimizar y posicionar tu pagina web.

html

Sirve exactamente para hacer de tu página web un proyecto con estilo propio, podrás colocar imágenes, textos, videos y todo lo que necesites.

css

Es un lenguaje enfocado a definir, crear y mejorar la presentación de un documento basado en HTML.

javascript

Permite hacer operaciones complejas en una página web. Muestra actualizaciones de contenido, animaciones gráficas entre otras acciones.
